# 用户使用手册

"学生课外辅导课时管理系统"帮助培训机构教师管理学生课程、排课、消课和课时统计。

---

## 1. 快速开始：首次使用流程

首次打开小程序，按以下步骤操作：

```
创建课程 → 设置排课 → 等待自动消课（或手动消课） → 查看统计
```

**详细步骤：**

1. 打开小程序，进入「课程」页面
2. 点击「+ 添加课程」，填写学生和课程信息
3. 进入「排课」页面，设置每周固定上课时间
4. 系统按排课计划自动消课（每 30 分钟检查一次）
5. 随时在「统计」页面查看课时使用情况

---

## 2. 创建课程

进入「课程」页面，点击「+ 添加课程」。

### 表单字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 学生姓名 | 是 | 学生真实姓名，便于查找 |
| 课程名称 | 是 | 如"数学辅导"、"英语口语"等 |
| 总课时数 | 是 | 购买的总课时数量（正整数） |
| 单价（元） | 否 | 每课时费用，用于费用统计 |
| 开课日期 | 是 | 课程开始日期 |
| 到期日期 | 否 | 课程有效截止日期，到期后无法消课 |
| 备注 | 否 | 补充信息，如学生特殊需求等 |

### 操作提示

- 总课时数填写后不可修改（已消课时会影响准确性），如需调整请联系管理员
- 到期日期设置后，过期课程会标记为「已过期」并停止自动消课
- 创建成功后会自动跳回课程列表

---

## 3. 设置固定排课

进入「排课」页面，为已创建的课程设置每周固定上课时间。

### 操作步骤

1. 点击「+ 添加排课」
2. 选择课程（从已创建的课程中选择）
3. 选择上课星期（周一至周日，可多选）
4. 确认添加

### 说明

- 每个课程可设置多个排课（如周一和周三各上一节）
- 排课设置后，系统会按计划在对应日期自动消课
- 如需删除排课，在排课列表中左滑删除

---

## 4. 手动消课

当需要立即消课或补录消课时，使用手动消课功能。

### 操作步骤

1. 进入「消课」页面
2. 选择要消课的课程
3. 确认消课日期（默认今天）
4. 点击「确认消课」

### 说明

- 同一课程同一天只能消课一次（幂等控制）
- 手动消课不受排课计划限制，可在任意日期操作
- 消课后会自动更新剩余课时，并记录操作日志
- 如果该日期已有自动消课记录，手动消课不会重复扣减

---

## 5. 查看日历和统计

### 5.1 日历视图

进入「日历」页面：

- 日期上的标记点表示当天有消课记录
- 点击具体日期，下方显示当天的消课详情
- 左右滑动切换月份
- 不同课程用不同颜色标记

### 5.2 统计图表

进入「统计」页面：

- **总览卡片**：总课程数、总课时、已消课时、剩余课时
- **课时消耗趋势图**：按周/月查看消课趋势
- **课程排名**：按已消课时排序，识别高频课程
- **费用统计**：根据单价计算已消耗费用（如设置了单价）

---

## 6. 预警说明

系统自动检测并在首页提示以下预警：

### 6.1 低课时预警

- 当某课程剩余课时 <= 3 时触发
- 在课程列表中以橙色标签显示「即将用完」
- 建议及时与家长沟通续费

### 6.2 即将过期预警

- 当课程到期日期在 7 天内时触发
- 在课程列表中以红色标签显示「即将过期」
- 过期后课程状态变为「已过期」，自动消课停止

### 6.3 已过期课程

- 超过到期日期的课程自动标记为「已过期」
- 已过期课程无法进行消课操作
- 可在课程详情中查看历史消课记录

---

## 7. 操作日志

进入「我的」→「操作日志」，查看所有操作记录。

### 记录内容

| 字段 | 说明 |
|------|------|
| 操作时间 | 操作发生的精确时间 |
| 操作类型 | 创建课程 / 删除课程 / 手动消课 / 自动消课 / 编辑排课等 |
| 操作详情 | 具体操作内容摘要 |
| 关联课程 | 涉及的课程名称 |

### 用途

- 追溯消课记录，解决争议
- 审核自动消课是否正确
- 追踪课程变更历史

---

## 8. 设置页功能说明

进入「我的」页面：

| 功能 | 说明 |
|------|------|
| 个人信息 | 显示当前登录用户信息 |
| 操作日志 | 跳转到操作日志页面 |
| 数据导出 | 导出课程和消课数据（如有实现） |
| 关于 | 显示版本信息 |
| 清除缓存 | 清除本地缓存数据，不影响云端数据 |

## 9. 常见问题 FAQ

### Q1: 为什么自动消课没有生效？

**可能原因：**
1. 课程未设置排课 → 前往「排课」页面添加
2. 课程已过期 → 检查课程到期日期
3. 当天已消课 → 同一天不会重复消课
4. 定时触发器异常 → 联系管理员检查

### Q2: 消课记录有误，如何撤回？

目前系统不支持撤回消课。如需修正：
1. 前往「操作日志」确认错误记录
2. 联系管理员手动处理（可在数据库中调整）

### Q3: 如何添加多个学生同一门课程？

每个学生需要单独创建课程。例如：
- 学生A-数学辅导（总课时20）
- 学生B-数学辅导（总课时20）

### Q4: 课时用完后怎么办？

1. 为该学生创建新课程（新的课时包）
2. 在原课程备注中标注「已续费」

### Q5: 如何查看某学生的全部消课记录？

1. 进入「课程」页面，点击具体课程
2. 课程详情页展示该课程的所有消课记录
3. 也可在「日历」页面按日期查看

### Q6: 可以一次性消多节课吗？

不可以。每次消课操作固定扣除 1 课时。如需扣除多节，需执行多次手动消课，或等待自动消课按排课计划执行。

### Q7: 换手机后数据还在吗？

在。所有数据存储在微信云开发数据库中，登录同一微信账号即可访问全部数据。本地缓存丢失不影响云端数据。

### Q8: 排课设置后可以修改吗？

可以。在「排课」页面删除原有排课，重新添加新的排课计划即可。已产生的消课记录不受影响。
